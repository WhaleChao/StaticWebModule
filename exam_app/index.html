🖥️ 方法一：主機端 Python 匯入 (更新全站)

  1. 將新的試題 PDF 和答案 PDF 放入 K:\EXAM\考畢題答\
  2. 檔名格式請遵循現有命名規則：
    試題：115110_0101_綜合法學(一)(憲法.pdf
    答案：115110_ANS0101_綜合法學(一)(憲法.pdf
  3. 在 PowerShell 中執行匯入腳本：
    python K:\MAGI\import_new_exam.py
  4. 匯入完成後推送至 GitHub：
    cd K:\StaticWebModule && git add -A && git commit -m "import new exams" && git push origin master
